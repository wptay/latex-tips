For example, we can reference \cref{sect:General} and equation \cref{eq:Lambda_SGLR} from the main paper but \cref{eq:f} is from this supplementary material. Similarly, we can cite \cite{LenTayQui:J16}, which is from the bibliography of the main paper.

\end{align}
